Eac3 Settings

attenuation_control: Option<Eac3AttenuationControl>

When set to attenuate3Db, applies a 3 dB attenuation to the surround channels. Only used for 3/2 coding mode.

bitrate: f64

Average bitrate in bits/second. Valid bitrates depend on the coding mode.

bitstream_mode: Option<Eac3BitstreamMode>

Specifies the bitstream mode (bsmod) for the emitted E-AC-3 stream. See ATSC A/52-2012 (Annex E) for background on these values.

coding_mode: Option<Eac3CodingMode>

Dolby Digital Plus coding mode. Determines number of channels.

dc_filter: Option<Eac3DcFilter>

When set to enabled, activates a DC highpass filter for all input channels.

dialnorm: i32

Sets the dialnorm for the output. If blank and input audio is Dolby Digital Plus, dialnorm will be passed through.

drc_line: Option<Eac3DrcLine>

Sets the Dolby dynamic range compression profile.

drc_rf: Option<Eac3DrcRf>

Sets the profile for heavy Dolby dynamic range compression, ensures that the instantaneous signal peaks do not exceed specified levels.

lfe_control: Option<Eac3LfeControl>

When encoding 3/2 audio, setting to lfe enables the LFE channel

lfe_filter: Option<Eac3LfeFilter>

When set to enabled, applies a 120Hz lowpass filter to the LFE channel prior to encoding. Only valid with codingMode32 coding mode.

lo_ro_center_mix_level: f64

Left only/Right only center mix level. Only used for 3/2 coding mode.

lo_ro_surround_mix_level: f64

Left only/Right only surround mix level. Only used for 3/2 coding mode.

lt_rt_center_mix_level: f64

Left total/Right total center mix level. Only used for 3/2 coding mode.

lt_rt_surround_mix_level: f64

Left total/Right total surround mix level. Only used for 3/2 coding mode.

metadata_control: Option<Eac3MetadataControl>

When set to followInput, encoder metadata will be sourced from the DD, DD+, or DolbyE decoder that supplied this audio data. If audio was not supplied from one of these streams, then the static metadata settings will be used.

passthrough_control: Option<Eac3PassthroughControl>

When set to whenPossible, input DD+ audio will be passed through if it is present on the input. This detection is dynamic over the life of the transcode. Inputs that alternate between DD+ and non-DD+ content will have a consistent DD+ output as the system alternates between passthrough and encoding.

phase_control: Option<Eac3PhaseControl>

When set to shift90Degrees, applies a 90-degree phase shift to the surround channels. Only used for 3/2 coding mode.

stereo_downmix: Option<Eac3StereoDownmix>

Stereo downmix preference. Only used for 3/2 coding mode.

surround_ex_mode: Option<Eac3SurroundExMode>

When encoding 3/2 audio, sets whether an extra center back surround channel is matrix encoded into the left and right surround channels.

surround_mode: Option<Eac3SurroundMode>

When encoding 2/0 audio, sets whether Dolby Surround is matrix encoded into the two channels.
